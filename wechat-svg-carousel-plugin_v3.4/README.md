# 微信公众号 SVG 轮播插件

这是一个简单的 Chrome 插件，旨在帮助微信公众号运营者在后台编辑器中快速添加图片轮播组件。该组件基于 SVG 动画原理实现，无需外部 JS 即可在微信文章中正常播放。

## 功能特点
- **SVG 动画实现**：利用 SVG 原生动画，绕过微信编辑器的 HTML/CSS 限制。
- **自定义配置**：支持自定义图片链接、宽高比和每张图片的停留时间。
- **一键插入**：在编辑器工具栏添加按钮，点击即可配置并插入。

## 安装步骤
1. 下载并解压 `wechat-svg-carousel-plugin.zip`。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择解压后的文件夹。

## 使用方法
1. 登录微信公众号后台，进入图文编辑页面。
2. 在编辑器顶部的工具栏中，你会看到一个新的“图片轮播”图标（绿色背景）。
3. 点击图标，在弹出的窗口中输入图片 URL（每行一个）。
4. 设置宽高比（如 16:9）和停留时间。
5. 点击“确定插入”，轮播组件将自动插入到当前光标位置。

## 注意事项
- **图片链接**：建议使用已上传到微信公众号素材库的图片链接，以确保在所有设备上都能正常显示。
- **预览效果**：在编辑器中可能无法直接看到动画效果，请通过“预览”功能在手机上查看最终效果。
